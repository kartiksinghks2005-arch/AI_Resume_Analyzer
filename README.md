# AI Resume Analyzer 🚀

AI Resume Analyzer is a Streamlit-based web application that analyzes resumes and provides intelligent feedback.

## Features
- Upload Resume (PDF / DOCX / TXT)
- Skill Detection
- Resume Strength Score
- Job Role Matching
- Missing Skills Analysis
- Resume Section Validation
- Word Cloud Visualization
- AI Resume Feedback using LLM

## Tech Stack
- Python
- Streamlit
- NLP
- WordCloud
- Groq API (LLM)

## Run Locally

Clone the repository:

git clone (https://kartiksinghks2005-arch-ai-resume-analyzer-app-aa594f.streamlit.app/)

Install dependencies:

pip install -r requirements.txt

Run the app:


streamlit run app.py
