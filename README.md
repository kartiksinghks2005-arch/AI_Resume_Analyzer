# 🚀 AI Resume Analyzer PRO

An AI-powered resume screening platform that analyzes resumes, detects skills, predicts job roles, and ranks candidates automatically.

---

## 🔍 Features

* Resume parsing (PDF, DOCX, TXT)
* Skill extraction using NLP
* Automatic job role prediction
* Resume strength scoring
* Missing skill detection
* AI-generated resume feedback
* Candidate ranking based on job description
* Interactive dashboard with visual insights

---

## 🧠 Technologies Used

* Python
* Streamlit
* NLP (Sentence Transformers)
* Groq LLM API
* Matplotlib
* WordCloud

---

## 📊 Key Capabilities

### 1️⃣ Resume Analysis

Extracts text and identifies technical skills from resumes.

### 2️⃣ Job Role Prediction

Predicts suitable roles such as:

* Data Scientist
* ML Engineer
* Data Analyst

### 3️⃣ Candidate Ranking

Ranks multiple candidates based on job description matching.

### 4️⃣ AI Resume Feedback

Uses LLM to generate professional resume improvement suggestions.

## 🖥 Example Dashboard

Upload resumes and receive:

* Skill detection

* Resume score
